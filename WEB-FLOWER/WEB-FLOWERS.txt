ПРОЄКТ: Telegram WebApp для квіткового магазину (Каталог + Замовлення + Адмінка)

ЦІЛЬ
Створити Telegram WebApp для квіткового магазину, де клієнти можуть переглядати каталог букетів і оформлювати замовлення з доставкою або самовивозом. Оплата здійснюється через LiqPay або готівкою при отриманні. Власник має адмін-панель для керування товарами і замовленнями.

ПЛАТФОРМА
Telegram WebApp (відкривається через кнопку в Telegram-боті)

КОРИСТУВАЧІ
1) Клієнт (без реєстрації)
2) Адміністратор (власник магазину)

---------------------
ФУНКЦІЇ ДЛЯ КЛІЄНТА
---------------------

1. Головна сторінка (Каталог)

Показати список букетів у вигляді карток:

Кожна картка містить:
- Фото букету
- Назву
- Короткий опис
- Ціну
- Кнопку "Детальніше"
- Кнопку "Замовити"

Фільтри:

- За ціною
- За типом квітів (троянди, тюльпани, мікс)
- За кольором (червоні, білі, рожеві, мікс)

Пошук по назві букету


---------------------

2. Сторінка товару

Показати:

- Велике фото
- Декілька фото
- Назву
- Повний опис
- Ціну

Опції:

Розмір букету:
- Малий
- Середній
- Великий

Розмір змінює ціну.

Додатково:

- Листівка (+50 грн)
- Шоколад (+150 грн)
- Іграшка (+200 грн)

Поле:

- Текст для листівки

Кнопка:

Додати в кошик


---------------------

3. Кошик

Показати:

- Список товарів
- Обрані опції
- Кількість
- Ціну

Можливість:

- Видалити товар
- Змінити кількість

Показати:

- Сума товарів
- Доставка
- Загальна сума

Кнопка:

Оформити замовлення


---------------------

4. Оформлення замовлення

Поля:

- Ім'я
- Телефон

Тип замовлення:

- Самовивіз
- Доставка

Якщо доставка:

Показати поля:

- Місто
- Вулиця
- Будинок
- Квартира
- Коментар

Вибір дати доставки

Вибір часу доставки

Кнопки оплати:

- Оплатити через LiqPay
- Оплата при отриманні


---------------------

5. Оплата LiqPay

Після натискання:

"Оплатити через LiqPay"

Система:

- Створює замовлення
- Генерує оплату LiqPay
- Перекидає користувача на сторінку оплати LiqPay

Після успішної оплати:

Показати:

Замовлення успішно оформлено

Показати:

Номер замовлення

Статус:

Оплачено


---------------------

6. Підтвердження замовлення

Показати:

Замовлення прийнято

Номер замовлення

Статус:

Новий або Оплачено

Кнопка:

Написати менеджеру в Telegram


---------------------

АДМІН ПАНЕЛЬ
---------------------

1. Вхід

Вхід по:

- Логін
- Пароль


---------------------

2. Замовлення

Список замовлень:

Показати:

- Номер замовлення
- Ім'я
- Телефон
- Сума
- Статус
- Дата

Статуси:

- Новий
- В роботі
- Готовий
- Доставлено
- Скасовано

При відкритті замовлення:

Показати:

- Дані клієнта
- Адресу
- Товари
- Опції
- Суму

Можливість:

- Міняти статус


---------------------

3. Товари

Адміністратор може:

- Додавати букети
- Редагувати букети
- Видаляти букети

Поля товару:

- Назва
- Опис
- Ціна
- Фото
- Категорії
- Колір
- Тип квітів

Перемикач:

Активний / Неактивний


---------------------

4. Налаштування

Адміністратор може змінювати:

- Телефон магазину
- Telegram посилання
- Адресу магазину

Доставка:

- Фіксована ціна доставки

Наприклад:

100 грн
